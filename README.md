Smart Medicine Dispenser System Using Arduino UNO

Description
This project is an Arduino-based smart medicine dispenser system designed to help users take medicines on time. The system can provide reminders and automate medicine dispensing using electronic components and programmed timing logic.

Components Used
- Arduino UNO
- RTC Module
- Servo Motor
- Buzzer
- LCD Display
- Push Buttons

Features
- Medicine reminder alert
- Automatic dispensing mechanism
- Time-based operation
- User-friendly system

Technologies Used
- Arduino IDE
- Arduino Programming
